# Dre Grace-Hannon, M.Ed., LPC - Professional Website

Professional website for Dre Grace-Hannon, Licensed Professional Counselor specializing in therapy for teens, adults, and couples in Texas.

## Website Structure

This Jekyll-based website includes:

- **Home/About** - Professional biography and therapeutic approach
- **Services** - Specializations including anxiety therapy, depression counseling, trauma treatment, PTSD therapy, couples counseling, and grief therapy
- **Insurance & Fees** - Insurance providers accepted and payment information
- **Contact** - Appointment scheduling and contact information

## Key Features

- Fully SEO-optimized for mental health and counseling search terms
- Mobile-responsive design
- Clear navigation for potential clients
- Direct links to online appointment booking
- Professional, warm, and accessible tone

## Running Locally

To preview the website locally:

1. Install dependencies:
   ```bash
   bundle install
   ```

2. Run the Jekyll server:
   ```bash
   bundle exec jekyll serve
   ```

3. View the site at `http://localhost:4000`

## Deployment

This site is configured for GitHub Pages deployment. Any changes pushed to the master branch will automatically build and deploy.
